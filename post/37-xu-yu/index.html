【37】絮语

· 2015-09-12 · # 基诺 # 基诺朱 # 37 # 反叛的鲁路修 # 枢木朱雀 # 同人 # 文章

“这是……钢琴？”指尖轻轻摩擦过洁白的漆面，朱雀轻声问道。

“没错，是莫妮卡的。”基诺以一副懒散的姿态斜靠在钢琴上，托着腮露出一个灿烂的笑容，“那家伙很喜欢弹钢琴，所以在伊露巴鲁宫也摆了一架。”

“原来是库鲁杰夫斯基卿的，难怪，”朱雀若有所思地点点头，“她真是位很有格调的女性。”

“哈哈，你可不要被她清纯的外表给骗了，那家伙发起飙来可是很恐怖的喔。”基诺挑了挑眉，正欲再说些什么，却被一旁的粉发少女打断了。

“但是办事靠谱，”阿尼娅摆弄着手机，头也不抬地说，“不像基诺，只想着玩。”

“哈哈，我哪有。”基诺干笑了两声，仗着身高优势在那柔软蓬松的粉发上胡乱揉了两把。阿尼娅用不悦的眼神看了基诺一眼，举起手机记录下了他恶作剧得逞后毫无形象地得意大笑的瞬间。

朱雀站在一旁，静静望着笑闹的两人。

“你们可别小看我，我也是很擅长钢琴的，虽然一点都不喜欢。”基诺自夸一般笑道。

“不喜欢为什么要学？”阿尼娅面无表情地问。

“尊敬的阿尼娅·阿尔斯托莱姆小姐，请问您觉得会弹钢琴的和空有一身蛮力的男人，哪一种更加迷人呢？”

“当然是挥剑杀敌的。”歪着头思考了片刻，阿尼娅认真地回答。

“这种审美是不被大众认可的！”基诺站直了身体，像是被踩了尾巴一般令人意外地认真了起来，“在欣赏了我的表演后，可还没有不会心动的人呢。”

他径直在钢琴前坐下，掀开了钢琴盖，修长的十指轻轻搭在琴键上，朝站在一边的阿尼娅抛去一个志得意满的眼神，随即将目光移到了琴键上。

随着手指轻敲在琴键上，琴槌击打琴弦奏出了第一个音符。

黑色的骑士手套映在黑白相间的琴键上，衬得手指更加修长。那双手灵巧地在琴键上翻动飞舞，一串悠扬的乐曲悄然弥散在空气中。

朱雀潜意识里觉得此时的基诺与方才的简直判若两人，但又说不出有什么不同。

少年修长的身躯挺得笔直，被包裹在量身定制的骑士制服中，勾勒出紧实的肌肉轮廓。金色卷曲的发尾随着头部的动作轻轻颤动，仿佛在显示它们的主人是多么沉醉于这场演奏之中。

那乐曲是极慢的，随着时间的推移渐渐加快，如同一张缤纷的画布逐渐铺开展现在面前。

眼前仿佛出现了幻影。那影像是极模糊的，亦或是被他封存在记忆深处绝对不想提起的禁忌。

少女朝他露出一个憔悴的微笑，苍白染血的手无力地伸向他：“……朱雀，你要去学校喔。”

身材高大的中年男人暴怒地大吼着什么，视野剧烈地晃动了几下后，映入眼帘的是铺天盖地的红和男人颓然倒地的残影。

年少时期那个短暂出现在他生命中的友人有一张唱片，他曾经偷偷潜入父亲的办公室偷来他珍藏的留声机，然后悄悄躲在神社深处的废弃仓库里听那美妙的旋律。

“真好听。”栗色卷发的女孩坐在轮椅上静静聆听，脸上露出了陶醉的神情，“令我想起了妈妈的摇篮曲。”

朱雀半弓着腰盘腿坐在地上，专注地盯着缓缓转动的唱片。

记忆中没有母亲的样子，似乎是在他很小的时候就去世了。而父亲也鲜少提及亡妻，在朱雀眼中，她是个满身谜团的女人。

但却意外地令人安心。仿佛他正安稳地躺在母亲怀里，耳边传来一把温柔的嗓音轻轻哼唱的摇篮曲，心中的躁动不安悉数被安抚平静了下来。

“这首曲子，叫什么名字？”

“是……喔。”黑发的少年微笑着望向他，温柔的眸中闪烁着点点星光。

……奇怪？

那首曲名像是碑文上被强行凿去的一块，不管他怎么回忆都是徒劳。那些丝丝缕缕的记忆如同随风飘散的轻烟般拂过他耳边，伸手去挽留，指尖却空空落落，什么也不曾抓住。

然而那份令人安心的感觉却留了下来，安抚着他的内心。一直以来背负着沉重负担的神经松弛了下来。他轻轻牵动嘴角，露出一个微不可察的笑容。

“阿尼娅快看，朱雀笑了！”琴声戛然而止。朱雀回过神来，发现基诺正笑嘻嘻地望着他，“这次打赌是我赢了哦，晚餐要请我吃圣代。”

朱雀像是没反应过来一样眨了眨眼，随即被基诺的笑容感染也跟着笑起来，抚平的眉峰和微弯的眼梢看上去暖意融融。

“谢谢，”他轻声道，“让你们担心了。”
